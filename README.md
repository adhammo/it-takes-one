# It Takes One

Puzzles and combat game based on cooperation theme.

## Machine

- Windows 10
- Unity 2021.3.2f1

## Packages

- URP
- Cinemachine
- Pro Builder
- Input System

## Ideas

- single player - team - puzzle/combat
- cartoony
- arabic support
- sketchfab
- kenney lowpoly
- poly haven
